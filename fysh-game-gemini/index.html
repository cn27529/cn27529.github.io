豐原高中 資訊研究社與Google Gemini 社團協作成果

響應「Google Gemini 全台高中社團祭」 [[2](https://rsvp.withgoogle.com/events/gemini-for-clubs/)]， 我們是來自臺中市立豐原高中的資訊研究社 [[3](https://fysh.tc.edu.tw/)]。 我們利用 Gemini 這位「AI 神社員」，協助我們進行社團活動企劃、文案撰寫及程式開發輔助。

成果說明

在這次的協作中，我們將 Gemini 應用於多個方面。我們利用它來「腦力激盪企劃」 [[2](https://rsvp.withgoogle.com/events/gemini-for-clubs/)]， 為我們的年度成果展發想了許多創意點子。

此外，Gemini 在「製作文案」 [[5](https://www.facebook.com/groups/2294169410829101/posts/4282788468633842/)] 方面也給了我們極大幫助，社團介紹和活動宣傳文案都變得更加吸引人。 身為資訊社，我們也嘗試讓 Gemini「協助寫程式」 [[4](https://rsvp.withgoogle.com/events/gemini-for-clubs/forms/registration)]， 它幫助我們除錯並優化了一個社團用的 Python 報名小工具。

  • 協作項目一： 成果發表會活動企劃。
  • 協作項目二： 社交媒體宣傳文案撰寫。
  • 協作項目三： Python 報名小工具程式碼優化。
