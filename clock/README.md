# 天干地支時鐘

## 簡介

天干地支時鐘是一個融合中國傳統曆法與現代時間顯示的互動式時鐘，將傳統的60甲子、12地支與24小時制巧妙結合在一個視覺化界面中，展現了中華文化的時間智慧。

## 核心特性

### 1. **多層時間系統**

- **外圈：60甲子刻度** - 代表傳統的60個時辰單位
  - 每刻度對應現代時間的1分鐘或1秒鐘
  - 高亮顯示當前的甲子分鐘和甲子秒數
- **中圈：12地支刻度** - 代表傳統的12時辰
  - 每個時辰對應現代時間的2小時
  - 高亮顯示當前的地支時辰
- **內圈：24小時刻度** - 現代24小時制
  - 旋轉180度對齊地支時辰系統
  - 只在奇數位置顯示數字，保持界面清晰

### 2. **動態太極圖**

- **中心太極圖**：每秒旋轉一圈，象徵時間的永恆流動
- **陰陽互動**：黑中有白，白中有黑，體現陰陽相生的哲學思想
- **精確定位**：完美對齊時鐘中心點，旋轉流暢自然

### 3. **即時時間顯示**

- **現代時間**：顯示當前日期與24小時制時間
- **干支時間**：顯示當前的地支時辰、甲子分鐘、甲子秒數
- **刻鐘系統**：顯示傳統的刻鐘（每15分鐘為一刻）

## 技術特點

### 視覺設計

- **響應式布局**：適應不同屏幕尺寸
- **漸層背景**：使用紫色漸層營造神秘氛圍
- **光影效果**：運用陰影和光暈增強立體感
- **色彩編碼**：
  - 🔴 紅色：秒數高亮
  - 🔵 藍色：分鐘高亮
  - 🟢 綠色：時辰高亮
  - 🟠 橙色：小時高亮

### 互動功能

- **即時更新**：每秒自動刷新時間顯示
- **視覺反饋**：當前時間對應的刻度會高亮顯示
- **懸停效果**：部分元素在鼠標懸停時有視覺變化

## 文化意義

### 時間哲學

1. **循環觀念**：60甲子、12地支構成循環的時間觀
2. **陰陽平衡**：太極圖體現了對立統一的哲學思想
3. **天人合一**：將天干地支與自然時間相結合

### 傳統與現代融合

- 保留了傳統的60甲子記時法
- 結合了現代的24小時制
- 以視覺化方式呈現抽象的時間概念

## 技術架構

### 前端技術

- **SVG**：所有圖形元素均為矢量圖形
- **CSS3動畫**：實現流暢的視覺效果
- **響應式設計**：適應不同設備

### 數據管理

- **Vue Pinia**：狀態管理
- **計算屬性**：實時計算干支時間
- **響應式更新**：每秒自動刷新

## 使用說明

### 時間解讀

1. **地支時辰**：每2小時為一個時辰（如：子時為23:00-01:00）
2. **甲子分鐘**：每1分鐘對應一個甲子組合
3. **甲子秒數**：每1秒鐘對應一個甲子組合

### 界面元素

- **時鐘主體**：包含所有時間刻度
- **信息面板**：顯示當前時間和圖例
- **太極圖**：中心旋轉的視覺焦點

## 設計理念

### 美學原則

- **對稱平衡**：所有元素對稱分布
